## Installation

1. Clone the repo

   ```sh
   git clone https://github.com/Ekansh38/Rust-Physics-Engine.git

   ```

2. Install Dependencies

   ```sh
   cargo install macroquad
   ```

3. Run the project
   ```sh
   cd main
   cargo run
   ```
